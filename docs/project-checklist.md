
# Checklist completa (sistema definito)

## Tracking
- Email: sent, replied, no_reply (5 giorni), bounced
- Calls: inbound/outbound/ended
- Meetings: booked (email/phone), canceled
- Deals: opened, won, lost
- Campaigns: started/paused/completed
- Products/services: interested, sold + ranking “hot”

## Aggregazioni
- period: daily/weekly/monthly
- agent_id
- campaign_id
- product_service_id
